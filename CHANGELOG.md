# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2024-XX-XX

### Added
- Custom cursor system with hover effects and click pulse
- Keyboard shortcuts (M, Space, Arrow keys)
- Smooth audio fade transitions (3s fade-in, 0.9s fade-out)
- Volume slider with real-time control
- Glassmorphism UI design
- 7-layer effects system (glow, particles, color grading, noise, vignette, depth, lighting)
- Mobile responsive design
- Touch device fallback for cursor
- CONTRIBUTING.md
- CODE_OF_CONDUCT.md
- SECURITY.md
- CHANGELOG.md
- GitHub issue templates
- Pull request template

### Changed
- Removed MP3 dependency — audio now embedded in video
- Removed Play/Pause button (intentional design choice)
- Improved progress stability and event handling
- Updated FiveM event handling
- Enhanced README with SEO optimization and badges

### Fixed
- Audio not starting immediately on some browsers
- Progress stuck at 0% in certain conditions
- Native cursor appearing occasionally on Windows
- Status text not updating properly

## [1.1.0] - 2024-XX-XX

### Added
- Glassmorphism UI design
- 7-layer effects system
- Improved mobile responsiveness
- Better progress interpolation

### Changed
- Updated CSS animations
- Improved audio handling

## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- Basic loading screen functionality
- FiveM loadProgress sync
- Video background
- Audio controls
- Custom cursor
